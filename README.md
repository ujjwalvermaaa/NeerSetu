# NeerSetu - Smart Community Health Monitoring System

## Overview
NeerSetu is a comprehensive water quality monitoring and health management system designed for rural communities. It combines IoT sensors, machine learning, and real-time analytics to provide early warning systems for water contamination and health risks.

## Features
- 🌊 Real-time water quality monitoring
- 🤖 AI-powered health risk prediction
- 📱 Mobile app for community access
- 📊 Interactive dashboard for administrators
- 🔔 Multi-channel alert system
- 🗺️ Geographic mapping and visualization

## Tech Stack
- **Frontend**: React, React Native, Tailwind CSS
- **Backend**: Node.js, Express.js
- **Database**: MongoDB, Firebase Firestore
- **ML Services**: Python, FastAPI, scikit-learn
- **IoT**: Python-based sensor simulation
- **Maps**: Leaflet, OpenStreetMap
- **Notifications**: Firebase, Twilio

## Quick Start
1. Clone the repository
2. Run `./start-all.sh` (Linux/macOS) or `start-all.bat` (Windows)
3. Access the dashboard at http://localhost:3000
4. Access the mobile app at http://localhost:3001
